# 👾 PacMan

A fully playable Pac-Man clone built in JavaScript using p5.js and p5.play — complete with a hand-crafted maze, 8 AI-controlled ghosts, dot collection, a lives system, and score tracking.

🔗 **Live Demo:** [Play Here](https://rayhasgor07.github.io/PacMan/)

---

## Features

### 🟡 Pac-Man
- Animated Pac-Man sprite with directional rotation
- Smooth movement with arrow keys or WASD
- Screen wrapping — exit one side, enter the other

### 👻 8 AI Ghosts
- 4 ghost types: Red, Pink, Blue, and Yellow
- Each ghost navigates the maze independently using zone-based AI with randomized direction changes at intersections
- Collision with a ghost costs a life and resets positions

### 🍬 Dot System
- Dots fill every valid path in the maze
- Collect all dots to win
- Sound effect plays on each dot collected

### ❤️ Lives System
- Start with 3 lives displayed as heart icons
- Lose a life each time a ghost catches you
- Game over screen when all lives are lost
- Win screen when all dots are collected

### 🔊 Audio
- Point collection sound
- Death/life lost sound

### 🔄 Restart
- Full reset button — restores lives, score, dots, and ghost positions
